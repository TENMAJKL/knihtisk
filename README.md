# Knihtisk

Konečně jazyk pro uživatele knihtisku. Nyní můžeme psát kvalitní kód ve staročeštině.

## Ukázka

```knihtisk
Vyřkni "krleš".
```

Výstup:
```c
#include<stdio.h>

int main(void) 
{
    printf("krleš");
}

```

A to jsou tak nějak jediné věci co tento jazyk (doufam zatím) umí.

## Kontribuce

Především je vítána jakákoliv znalost staročeštiny, takže pokud některé pojmy nejsou staročesky, merge requesty jsou vítány.
